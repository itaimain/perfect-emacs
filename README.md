Perfect Emacs
=============
perfect-emacs project is my own emacs configuration.

Install
----------
### Backup your existing configuration (Optional)
```bash
cd ~/
mv .emacs.d .emacs.d.backup
mv .emacs .emacs.backup
```

### Clone perfect-emacs
```bash
git clone https://github.com/itaimain/perfect-emacs ~/.emacs.d
```
